# 1. Create the NestJS Project

```
nest new [project name]
```

# 2. Install packages

```
npm install @nestjs/graphql 
            graphql-tools 
            graphql 
            @nestjs/apollo
            @apollo/server
```

# 3. Create Module

```
nest g module [module name]
```

`g` is for generate.
